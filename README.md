# Financial Fraud Detection Dashboard - Power BI

## Project Overview

This project showcases an interactive Power BI dashboard developed to analyze financial transaction data and identify fraud patterns. The dashboard provides meaningful business insights through dynamic visualizations, KPI cards, filters, and interactive reports.

---

## Features

- Interactive KPI Cards
- Fraud vs Genuine Transaction Analysis
- Transaction Type Analysis
- Transaction Amount Analysis
- Average Transaction Analysis
- Sender & Receiver Analysis
- State-wise Transaction Map
- Interactive Slicers and Filters
- Professional Dashboard Design
